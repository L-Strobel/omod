DummyLocation

data class DummyLocation(val coord: Coordinate, val latlonCoord: Coordinate, var odZone: ODZone?, val transferActivities: Set<ActivityType>) : LocationOption

Dummy location. Used for locations outside the model area (focus area + buffer area). These locations are introduced if an odFile is provided that contains TAZs that don't intersect the model area.

Parameters

coord

Coordinates of centroid in model CRS (Distance unit: meters)

latlonCoord

Coordinates of centroid in lat-lon

odZone

The od-zone associated with the location

transferActivities

Activities which can cause arrival and departure here. Only activities defined in the odFile can cause an agent to arrive at a dummy location or leave it.
