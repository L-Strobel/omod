class ActivityDataStore(activityGroups: List<ActivityGroup>)

Data structure for the activity chain probability distributions. Structure is designed for fast access.

For every combination of the features weekday, homogeneous group, mobility group, and age ActivityDataStore contains on element of ActivityGroup.
