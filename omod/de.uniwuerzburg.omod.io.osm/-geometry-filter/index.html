GeometryFilter

class GeometryFilter(area: Geometry, geometryFactory: GeometryFactory, idTrackerType: IdTrackerType, clipIncompleteEntities: Boolean, completeWays: Boolean, completeRelations: Boolean, cascadingRelations: Boolean) : AreaFilter

Osmosis filter. Returns all osm objects in a given area.

Parameters

area

Geometry of the area. Must be in EPSG 4326, i.e. lat lon coordinates.

geometryFactory

Factory for point creation in check. Precision should be consistent across an application

idTrackerType

ID tracker. See Osmosis documentation.

clipIncompleteEntities

Remove ways and relations that are only partly in the area

completeWays

Include all nodes of ways even if the node is outside the area

completeRelations

Include all nodes of relations even if the node is outside the area

cascadingRelations

Include all relations that are included in a relation that is intersects the area
